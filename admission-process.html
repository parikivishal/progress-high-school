Required Documents

Prepare these documents for a smooth admission process

Academic Records

  • • Previous year's report card
  • • Transfer certificate (if applicable)
  • • Character certificate
  • • Migration certificate (for inter-state)

Identity Proof

  • • Birth certificate (original + copy)
  • • Aadhaar card of student
  • • Passport size photos (6 copies)
  • • Medical fitness certificate

Address Proof

  • • Electricity/Water bill (latest)
  • • Rental agreement/Property papers
  • • Aadhaar card of parents
  • • Voter ID of parents

Financial Documents

  • • Income certificate
  • • Bank account details
  • • Salary slips (last 3 months)
  • • Caste certificate (if applicable)
Note: All documents should be self-attested. Original documents required for verification only.
